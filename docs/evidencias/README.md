# Evidencias do prototipo fisico

Inclua nesta pasta as fotos usadas para comprovar o hands-on da Sprint 03.

Arquivos recomendados:

```text
esp32-montagem.jpg
esp32-led-verde.jpg
esp32-led-vermelho.jpg
```

As imagens devem mostrar:

- ESP32 fisico;
- LED verde no GPIO 18;
- LED vermelho no GPIO 19;
- buzzer no GPIO 23;
- montagem completa e alimentada;
- funcionamento durante a validacao NFC no aplicativo.

Nao inclua senhas de Wi-Fi, documentos pessoais ou outros dados sensiveis nas fotos.
